题意：三种逻辑运算,and,or,xor
给出N个变量，取值0或1
M个运算式子，问是否有解

分析：建图非常经典，加深了对2-SAT模型构建的理解
2-SAT模型中，每个有向边x->y表示 选了x必选y

AND运算：
若 c == 1, x->x' y->y' 所以是自我矛盾，不会出现x,y为0的情况
若 c == 0, x'->y y'->x 表示至少有一个为0

OR运算：跟AND十分类似，略

XOR运算：
若 c == 1, x->y' x'->y y->x' y'->x
若 c == 2, x->y  y->x  x'->y' y'->x'

然后就是跑一遍可行性判断
